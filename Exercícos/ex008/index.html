Principais formatações

Negrito / destaque

Nesta frase temos negrito usando a tag B(não semântico)

Nesta frase temos um termo em destaque, usando a tag STRONG(semântico)

Itálico / ênfase

Nesta frase temos um termo em itálico usando a tag I(não semântico)

Nesta frase temos um termo em ênfase usando a tag EM(semântico)

Texto marcado

Também podemos fazer texto marcado usando a tag MARK

Esse parágrafo também tem marcação

Texto grande e pequeno

Neste parágrafo temos um texto grande e um texto pequeno

Texto excluído

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado.

Texto inserido

Podemos marcar um texto como inserido para dar ênfase e indicar que ele foi adicionado

Texto sobrescrito

Podemos usar essa formatação para coisas do tipo: 231 + 98

Texto subscrito

Podemos usar para coisas do tipo: H2O
